# Bundled force field

This repository includes **CHARMM36m + CGenFF** (folder name set in `mdprep/config.sh` → `FF_DIR`).

If you publish simulations using this force field, cite the original CHARMM36 and CGenFF papers and comply with the upstream license terms from the force field provider.

GmxKit does not modify the force field files; they are shipped for one-step installation convenience.
